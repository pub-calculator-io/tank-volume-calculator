The tank volume calculator finds the total volume of various tank shapes in gallons, liters, and cubic meters. It calculates the liquid volume of partially filled tanks.

[https://www.calculator.io/tank-volume-calculator/](https://www.calculator.io/tank-volume-calculator/)

== Usage ==

Add the Tank Volume Calculator shortcode to your page, post or sidebar:

`[ci_tank_volume_calculator]`

Add the following code to your template where you would like the Tank Volume Calculator to appear:

`<?php display_ci_tank_volume_calculator(); ?>`

== Installation ==

1. Upload the Tank Volume Calculator /ci_tank_volume_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Tank Volume Calculator plugin through the "Plugins" menu in WordPress.
